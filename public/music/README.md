### Option 2: Royalty-Free Music Sources

Here are some great sources for retro racing game music:

1. **Incompetech** (https://incompetech.com)

    - Search for "8-bit" or "chiptune" or "racing"
    - Free with attribution

2. **FreeMusicArchive** (https://freemusicarchive.org)

    - Filter by "Electronic" and "8-bit"
    - Various licenses available

3. **OpenGameArt** (https://opengameart.org)

    - Specifically for game music
    - Search for "racing" or "retro"

4. **Chiptune.com** (https://chiptune.com)
    - Dedicated chiptune music community
    - Many free tracks

### Option 3: Create Your Own

Use tools like:

-   **Beepbox** (https://beepbox.co) - Browser-based chiptune creator
-   **FamiTracker** - NES-style music tracker
-   **LMMS** - Free DAW with chiptune plugins

## File Format

The game supports:

-   MP3 (recommended for compatibility)
-   OGG (better quality, smaller file size)
-   WAV (uncompressed, larger files)

## Installation

1. Download or create your music files
2. Rename them to match the required filenames above
3. Place them in this directory (`public/music/`)
4. The game will automatically load and play them

## Volume Levels

The game has separate volume controls for:

-   Music (background tracks)
-   SFX (engine sounds, UI sounds)
-   Master (overall volume)

Music volume defaults to 30% to avoid overpowering the engine sounds.
